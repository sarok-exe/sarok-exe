# Hi, I'm Sarok

I build tools for the terminal and desktop. Arch Linux user, Niri compositor, Python/JS.

### Projects

- [**pictopress**](https://github.com/sarok-exe/pictopress) — Batch Image-to-WebP converter with Textual TUI. Navigate with arrow keys, select with Space, press Enter to compress.
- [**achem**](https://github.com/sarok-exe/achem) — Deep web research tool. Searches 100+ sources, scrapes content, generates AI summaries via OpenRouter/Groq/Gemini/Ollama.
- [**sarok-area**](https://github.com/sarok-exe/sarok-area) — Arch Linux dotfiles: Niri, Waybar, Rofi, Swaync, Kitty, GTK. Warm accent theming, compact/minimal UI.
- [**Al-Zaman**](https://github.com/sarok-exe/Al-Zaman) — Terminal clock displaying Gregorian & Hijri dates with big time display. Switch between 22 Arab countries.
- [**Mega-Video**](https://github.com/sarok-exe/Mega-Video) — Webapp to download any video using yt-dlp & ffmpeg.
